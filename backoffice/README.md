# Safinity-app
Backoffice, para a app mobile, desenvolvido em React para garantir a segurança em eventos públicos de grande escala.


## Tecnologias Utilizadas

- React 
- JavaScript / TypeScript
- MapBox API

---

## Estrutura do Projeto

```text

src/
├── @types/      # Componentes reutilizáveis
├── api/         # comunicação a MapBox
├── assets/      # Imagens, ícones, fontes
├── components/  # Componentes reutilizáveis
├── data/        # Ficheiros JSON
├── layout/      # layout de navbar e sidebar
├── pages/       # páginas
├── theme/       # theme
├── App.tsx      
├── index.css      
└── main.tsx 
```

---

## Pré-requisitos

- Node.js >= 18
- npm 


Instalar dependencias:
```
npm install
```
---

## Como Executar o Projeto

1. Instalar dependências:
```
npm install
```
2. Iniciar o projeto:
```
npx start
```
3. Executar no dispositivo:
- Abrir link de localhost

---

## Configuração de Ambiente

Cria um ficheiro `.env` em frontend/ :

''' 
EXPO_PUBLIC_MAPBOX_TOKEN=key_para_mapbox
'''
